Markdown template example

In the HTML template, be aware of the | safe Jinja filter which allows raw HTML to be renedered.

Everything after this paragraph will be read from a Markdown file.

{{ md.content | safe }}

This is the end of the Markdown file content.

This is the Markdown meta:

{{ md.meta | safe }}

You can use the Markdown meta in many ways. At the head of this HTML template, I used it to set the title of the page. Now the title shows up as the name of your browser tab.

I read the h4_color metadata key to apply a color to this header using a style tag.
